    • Background

    This is a tool that analyzes email text. The analyis displays the expressions conveyed by the email, including traits, such as anger, joy, confidence, and an indicator of how informal or formal the text is. The analysis alerts the user of possible cyber - bullying and harassment. It can reduce crime by 20 - 30%. The application is live at MixMax

    Method

    The stack uses Embedded JavaScript and IBM Watson Tone Analyzer, and MixMax APIs.
